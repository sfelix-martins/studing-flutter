# Contacts Manager

A Flutter application to manage contacts.

## Getting Started

This project is part of the course "Criação de Apps Android e iOS com Flutter - Crie 16 Apps" published on the Udemy platform.

## Features/Widgets

- Persistent Storage (Sqflite)
- Singleton
- Card
- CircleImage
- BottomSheet
- AlertDialog
- PopUpMenuBottom
- ImageBuilder
- ImagePicker
- UrlLauncher
- Focus
