# Giphy Share

A Flutter application to search and share GIFs.

## Getting Started

This project is part of the course "Criação de Apps Android e iOS com Flutter - Crie 16 Apps" published on the Udemy platform.

## Features/Widgets

- Navigator
- GridView
- ImageNetwork
- FadeInImage
- GestureDetector
- Share
- FadeInImage
- ProgressIndicator
- Import Files

## Plugins

- http: ^0.12.0+4
- share: ^0.6.3+5
- transparent_image: ^1.0.0
